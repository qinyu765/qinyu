# P3R 风格博客

一个基于 Persona 3 Reload 视觉风格的博客项目，技术栈为 React 19 + Vite + TypeScript。

## 快速开始

先确保已安装 Node.js 与 pnpm。

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```

## 部署

项目已配置 GitHub Actions 自动部署到 GitHub Pages，推送到 `main` 分支即可触发。
